## 🛰️ Ecosistema de Inteligencia (Proyecto AESIR)

### 🧿 MÍMIR V1: Engine de Inteligencia Estratégica
*Sistema de procesamiento y normalización de ADN de datos para toma de decisiones ejecutivas.*
- **Arquitectura:** Microservicios dockerizados en Railway con sincronización híbrida local-nube.
- **Data Hardening:** Algoritmos de limpieza heurística en Python/Pandas que transforman datos crudos (Excel/CSV) en insights operativos listos para IA.
- **Orquestación:** Integración avanzada con n8n y modelos de lenguaje (LLMs) para análisis de varianza automática y generación dinámica de reportes.

### 🌉 Bridge & Dashboard (Ares Protocol)
- **Skirnir (El Cerebro):** Centro de comando y auditoría para el monitoreo de agentes y métricas críticas (Actual vs Target).
- **Gungnir (La Lanza):** Motor de ventas inteligente con RAG (Pinecone) y Gemini 1.5 Flash, optimizado para conversión de alta fidelidad.
- **Interfaz:** Dashboards de baja latencia bajo estética de "Búnker Técnico" (Vanilla CSS / Dark Mode).

### ⚙️ Automatización y Soberanía
- **Protocolos de Sincronía:** Scripts en PowerShell Core (`MIMIR_N8N_ULTRA.ps1`, `Refresh_Mimir.ps1`) para el despliegue y actualización de flujos n8n.
- **Blindaje Técnico:** Implementación de seguridad multicapa (Basic Auth, Tokens X-Soberano) y redundancia local para continuidad operativa.
